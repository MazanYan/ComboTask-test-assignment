# ComboTask-test-assignment

This is a test assignment for position on ComboTask.
The application is fetching HackerNews by https://api.hnpwa.com (docs: https://github.com/tastejs/hacker-news-pwas/blob/master/docs/api.md)

Used technologies: Webpack, React, React-router, Babel, Redux.

# Install application

1. ```git clone https://github.com/MazanYan/ComboTask-test-assignment```
2. ```cd ComboTask-test-assignment```
3. ```npm i```
4. ```npm start```

To build application run ```npm run build```
The ourput will be located in ```dist``` directory.
